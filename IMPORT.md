كيفية تحويل ملف CSV/Excel للمعدات إلى SQL جاهز للاستيراد في Supabase

الملفات المضافة:
- `scripts/csv_to_seed.js` : سكربت Node.js يقرأ ملف CSV ويولد `seed_equipment.sql` يحتوي على أوامر INSERT لجدول `equipment`.

افتراضات:
- ملف CSV يكون UTF-8 وبه الأعمدة: الفئة, اسم المعدة, العدد, الحالة, ملاحظات (كما في الملف المرفق).
- Supabase يحتوي على الجدول `equipment` كما في `supabase.sql` (يجب تشغيله أولًا).
- لن نحتاج لمفاتيح Supabase لتوليد ملف SQL؛ ستستعمل ملف SQL الناتج في لوحة Supabase.

خطوات التشغيل (PowerShell):

1. افتح PowerShell في مجلد المشروع (`equipment-manager-web`).
2. شغّل السكربت مع مسار ملف CSV المرفوع. مثال:

```powershell
node .\scripts\csv_to_seed.js "C:\Users\duq13\Downloads\الجرد_النهائي_بعد_تحديث_الحالات.xlsx - Sheet1.csv"
```

3. سينشأ ملف `seed_equipment.sql` في نفس المجلد (إلا إذا حددت مسارًا آخر كمعامل ثاني).
4. افتح Supabase -> SQL Editor، والصق أو رفع محتوى `seed_equipment.sql` ثم شغّل.

ملاحظة أمان:
- لا تشارك مفاتيح المشروع (anon key أو service_role) هنا. إن أردت، أشرح لك خطوات استيراد SQL في لوحة Supabase خطوة بخطوة.

مشاكل معروفة:
- إن كان CSV يحتوي على فواصل داخل خانات أو تنسيقات مختلفة قد يحتاج السكربت تعديل بسيط. أستطيع تعديل السكربت ليتعامل مع تنسيقات أخرى إن احتجت.

إذا تريد، أستطيع توليد ملف SQL مباشرة الآن من الملف المرفق (لا يحتاج مفاتيح) وأرفعه هنا — أخبرني "ولد SQL من الملف" وسأولده وأضيفه للمشروع.
